## 二、详细需求

### R1：试题库

**描述**：所有用户可浏览系统中的试题文件列表。试题库按考试分类组织，每场考试对应一份试题文件。

**入口**：

1. **"我的考试"页面**
   - 考试卡片在考试时间结束后（`end_at` 到期）显示两个按钮：
     - 「浏览试题」— 在线预览试卷内容
     - 「下载」— 下载 docx 文件
   - 考试未结束时不显示这两个按钮

2. **独立"试题库"菜单**
   - 侧边栏新增"试题库"入口
   - 展示所有考试的试题文件列表
   - 不限制用户是否参加了该考试，所有用户均可查看
   - 列表支持按分类筛选

---

### R2：试卷自动生成与在线预览

**描述**：考试时间结束后，系统自动生成 docx 格式的试卷文件，支持在线浏览和下载。

**生成时机**：
- 考试的 `end_at` 时间到期后自动生成
- 不是归档(archive)时生成
- 生成任务异步执行，生成完成后标记状态

**docx 格式要求**：
- 清晰易读，适合打印和阅读
- 包含：考试名称、培训讲师、题目列表
- 每题包含：题号、题干、选项、正确答案标记
- 格式设计由开发团队决定，确保专业美观

**在线预览**：
- 网页版试卷预览页面
- 支持弹出大窗口展示试卷内容
- 内容与 docx 一致，以只读方式呈现

---

### R3：单场考试统计

**描述**：管理员在考试详情页查看单场考试的统计数据。

**入口**：考试详情页（现有"编辑"按钮进入的页面）新增一个 tab，名为"考试统计"。

**统计内容**：
- 参考总人数
- 及格人数
- 不及格人数
- 及格率
- 分数分布区间（如：0-59 / 60-69 / 70-79 / 80-89 / 90-100）
- 平均分、最高分、最低分

---

### R4：培训讲师

**描述**：考试信息中新增"培训讲师"字段，替代原有的"考试说明"。

**规则**：
- 可选字段，非必填
- 可选择多个培训讲师
- 从系统用户中选择，使用类似人员搜索的选择器
- 无权限限制，任何人都可以被选为讲师（包括自己）
- 删除原有的"考试说明"字段

**数据模型变更**：
- `Exam` 表删除 `description` 字段
- 新增关联表或 JSON 字段存储培训讲师列表

---

### R5：系统总览看板

**描述**：管理员专用的系统级数据汇总看板。

**权限**：仅管理员可见

**入口**：工具组下的独立工具

**数据卡片**：

| 指标 | 说明 |
|---|---|
| 考试场数 | 系统中所有考试总数 |
| 考试总参与人次 | 所有考试的参与人次总和 |
| 考试及格率 | 整体及格率 |
| 题目总数 | 系统中所有题目数量 |
| 试卷总数 | 系统中所有试卷数量 |

**图表**：

| 图表 | 类型 | 数据来源 |
|---|---|---|
| 题库题型分布 | 柱状图 | 单选/多选/判断/填空 数量 |

---

### R6：试卷库分类管理

**描述**：试卷库支持分类管理，考试可归属一个分类。

**分类规则**：
- 一级扁平结构（不支持子分类）
- 一个考试只能属于一个分类
- 管理员有权限管理分类（增删改）

**管理界面**：
- 新增"分类管理"页面
- 支持：新增分类、编辑分类名称、删除分类（需处理关联考试）

**考试关联分类**：
- 创建/编辑考试时，可选择所属分类
- 分类选择器为下拉框，展示所有分类

**试题库展示**：
- 试题库页面支持按分类筛选
- 分类标签展示在试题列表中
